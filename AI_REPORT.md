# Raport utilizare AI tools

## 1. Lovable AI
- Generat ~95% din codul frontend inițial
- Componente React, routing, integrare Supabase

## 2. Ollama + Llama3 (local)
- Motor pentru ambii agenți AI din aplicație
- Agent Antrenor: generează planuri de antrenament JSON
- Agent Macro: estimează calorii și macronutrienți din text liber
- Zero costuri API, date nu pleacă în cloud

## 3. Claude (claude.ai)
- Planificare arhitectură și structură repo
- Debugging și code review
- Generare documentație

## 4. ChatGPT
- Formulare user stories
- Generare criterii de evaluare pentru agenți

## Concluzie
Peste 90% din cod și documentație au fost generate cu ajutorul AI.
Agenții rulează complet local prin Ollama/Llama3.
