# Raport utilizare AI tools

## 1. Lovable AI
- Generat ~95% din codul frontend initial
- Componente React, routing, integrare Supabase

## 2. Google Gemini 2.5 Flash (cloud API)
- Motor pentru ambii agenti AI din aplicatie
- Agent Antrenor: genereaza planuri de antrenament JSON personalizate pe profilul utilizatorului
- Agent Macro: estimeaza calorii si macronutrienti din text liber, respectand formula Atwater
- Accesat direct din browser via Google AI Studio API

## 3. Claude (claude.ai)
- Planificare arhitectura si structura repo
- Debugging si code review
- Generare documentatie si README
- Migrare agenti de la Ollama la Gemini

## 4. GitHub Copilot / ChatGPT
- Formulare user stories
- Generare criterii de evaluare pentru agenti

## Concluzie
Peste 90% din cod si documentatie au fost generate cu ajutorul AI.
Agentii folosesc Google Gemini 2.5 Flash prin API cloud, fara costuri pentru utilizare in demo.
